Covid 19tracker

Covid 19 tracker

In this project I made a simple covid 19 tracker app that fetches data from the https://covid19api.com/ API. I had a fun time making this and solving some of the issues that occured during the development of this app

This API allows you to follow the progress of the coronavirus around the world. Data returned includes the number of new, active, critical, recovered, and total cases by country. You can also get historical COVID-19 data filtered by country.
